Win7 虚拟机安装配置

发表于 2019-08-06 更新于 2020-05-20 分类于 Windows 阅读次数： 评论次数：
本文字数： 3.4k 阅读时长 ≈ 3 分钟

整个 Win7 的虚拟机模板, 方便以后使用.

安装

VMware 下安装

简易安装. 版本 cn_windows_7_ultimate_with_sp1_x64_dvd_u_677408.iso. 分配了 32 GB 磁盘空间. 过程略.

激活

KMSpico (对 Win7 旗舰不起作用)

官网:

没有官网, 是发布在 https://forums.mydigitallife.net 论坛上的. 作者 heldigard. s1ave77 也是他的一个马甲.

搜索名为 KMS tools 的子论坛, 有个名为 KMSpico -Official Thread- 的帖子即为官方发布帖子. 上面有提供下载链接. 以下是帖子的一个 txt 附件, 内附下载信息.

这是他早前的另一个帖子 https://forums.mydigitallife.net/threads/how-to-kms-host-and-tools-v-3-0.39048.

KMSpico 运行需要 Microsoft .NET Framework 4 框架: https://www.microsoft.com/zh-cn/download/details.aspx?id=17718 .

它并没有说支持 Ultimate 版本, 但评论区有人成功了, 但我没成功.

Windows 7 Loader (成功)

官网: Windows 7 Loader by DAZ – 2019. 目前版本 2.2.2 . 下载要先填写一些问卷, 有点烦人. 本来是怕国内那些乱七八糟的软件网站下的软件有问题, 但官网提供了哈希, 不想填问卷就国内站点下载了对比哈希就行.

解压缩, 打开软件点一下 Install 再重启就安装完了.

注意: 有个网站叫 dazwindowsloader.com, 标题叫 Windows 7 Loader 2.6.2 By Daz Download [Windows 7 Activator], 长得有模有样的, 版本都出到 3.2 了, 谷歌搜索排名还很靠前. 这是假的, 会弹广告什么的. 最新版就是 2.2.2. 下面两张图中上面的是真的下面的是假的.

(甚至还搞了个假的 FEEDBACK??)

配置

开机启动小键盘

Win+R -> regedit -> HKEY_USERS\.DEFAULT\Control Panel\Keyboard -> InitialKeyboardIndicators, 双击把值改为 2 .

关闭 Windows Update

控制面板\系统和安全\Windows Update\更改设置. 免得给我升到 Win10 了...

必备软件

7-Zip | 火绒安全软件 | 火狐浏览器
