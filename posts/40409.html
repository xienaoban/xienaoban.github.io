美化软件

Gnome Tweak Tool: 美化必备

首先肯定是安装 gnome-tweak-tool.

Dash To Dock

是 Tweak 的一个扩展. 可以从 Ubuntu Software 搜索下载安装 Dash to dock, 然后就能从 Tweak 的 Extensions 标签页下找到它.

主要功能: Dock 图标居中显示; 将 Dock 的 Show Applications 从右侧 / 底部 挪到 左侧 / 上方.

也支持用指令. 比如要将 Show Applications 从 Dock 挪动位置可以输入:

Hide Top Bar

用于隐藏顶部栏.

可从 Ubuntu Software 搜索下载安装. 一样可以在 Tweak 的 Extensions 标签页下找到它.

User Themes

用于将主题配置文件放在用户文件夹, 以防止重装导致美化丢失.

一样是 Tweak 的扩展, 且可从 Ubuntu Software 搜索下载安装.

主题

Vimix Gtk Theme

点击前往官网. 我用的是 vimix-normal-dark 下的 vimix-dark-laptop-beryl.

Vimix Icon Theme

点击前往官网. 是 Vimix 的图标包.

下载后解压到 /usr/share/icons/ .
