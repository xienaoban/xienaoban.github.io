C / C++

GCC 工具: gcc -M <file>.cpp

对于 C++ 文件:

gcc -M <file>.cpp : 显示该文件的所有依赖.

gcc -MM <file>.cpp : 显示该文件的所有依赖 (不包括系统依赖的头文件).
